🎮 Live Multiplayer

Live multiplayer needs a free Firebase project (one-time, ~5 min). Until it's set up, the rest of the app works normally.

  1. Go to console.firebase.google.com → Add project.
  2. Build → Realtime Database → Create Database → start in Test mode.
  3. Project settings → Your apps → Web (</>), register an app, copy the config.
  4. Paste the values into firebase-config.js, then commit & push.

The web config is safe to commit — access is controlled by Database Rules.
