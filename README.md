# 🍺 The Tipsy Quizmaster — Pub Trivia

A single-file, zero-dependency pub trivia game that runs in any browser.

**▶️ Play it live:** https://jamiejacob100-svg.github.io/pub-trivia/

## Features
- 1–6 teams, pass-and-play
- 8 categories (General, Science, History, Geography, Music, Film & TV, Sport, Food & Drink)
- 64 multiple-choice questions, shuffled each game
- Choose 5/10/15/20 questions per game
- Optional per-question timer (15/20/30s or off)
- Live scoring, podium results, medals, and tie handling

## Running locally
Just open `index.html` in a browser — no build step, no install.

## How it's hosted
Served as a static site via GitHub Pages from the `main` branch.
